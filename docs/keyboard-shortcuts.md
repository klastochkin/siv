# Keyboard Shortcuts Reference

## File Operations
- `⌘O` - Open file
- `⌘W` - Close window
- `⌘Q` - Quit application

## Navigation
- `→` or `Space` - Next image
- `←` - Previous image

## Zoom Controls
- `⌘+` or `⌘=` - Zoom in
- `⌘-` - Zoom out
- `⌘0` - Fit to window (default)
- `⌘1` - Actual size (100%)

## View Options
- `I` - Toggle info bar
- `Esc` - Exit fullscreen or close window

## Mouse & Trackpad Gestures
- **Scroll wheel** - Zoom in/out (centered on cursor position)
- **Pinch gesture** - Zoom in/out
- **Click & drag** - Pan image when zoomed in

## Tips
- Images automatically scale to fit the window while maintaining aspect ratio
- Navigation wraps around (last image → first image)
- All images in the same folder are automatically included in navigation
- Pan is only available when zoomed in beyond fit-to-window scale
